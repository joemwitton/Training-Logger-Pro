
# Training Logger Pro

This project is a PowerShell-based Training Logger GUI developed as part of my apprenticeship (Zwischenstand).

## Purpose
The goal of this project is to log training sessions, analyse training load, track personal records (PRs) and generate reports.

## Current State
- Windows Forms GUI implemented in PowerShell
- Multiple tabs (Dashboard, History, Reports, Load, PRs, Charts, Settings)
- Training sessions can be logged and stored locally
- Load and PR views are implemented in the GUI

## Technologies
- PowerShell 7
- Windows Forms (.NET)
